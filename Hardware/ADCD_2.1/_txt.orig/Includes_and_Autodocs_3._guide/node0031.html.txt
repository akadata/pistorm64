amiga.lib/CxSignal ]" border=0> NAME CxSignal -- create a commodity signaller object. (V36) SYNOPSIS signalerObj = CxSignal(task,signal); CxObj *CxSignal(struct Task *,LONG); FUNCTION This function creates a Commodities signal object. The action of this object on receiving a Commodities message is to send the 'signal' to the 'task'. The caller is responsible for allocating the signal and determining the proper task ID. Note that 'signal' is the signal value as returned by AllocSignal() , not the mask made from that value. This function is a C-language macro for CreateCxObj() , defined in . INPUTS task - the task for the signaller to signal signal - the signal bit number for the signaller to send RESULTS signallerObj - a pointer to the signaller object, or NULL if it could not be created.
