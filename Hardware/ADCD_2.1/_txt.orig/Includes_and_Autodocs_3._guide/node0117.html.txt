(V44) SYNOPSIS ReleaseDTDrawInfo (o, handle); a0 a1 VOID ReleaseDTDrawInfo (Object *, APTR); FUNCTION This function is used to release the information obtained with ObtainDTDrawInfoA() . This function invokes the object's DTM_RELEASEDRAWINFO method using the opMember message structure. INPUTS handle - Pointer to an object returned by ObtainDTDrawInfoA() .
