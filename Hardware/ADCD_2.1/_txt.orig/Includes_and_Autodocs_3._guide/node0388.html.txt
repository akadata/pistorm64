input.device/IND_SETTHRESH ]" border=0> NAME IND_SETTHRESH -- Set the key repeat threshold FUNCTION This command sets the time that a key must be held down before it can repeat. The repeatability of a key may be restricted (as, for example, are the shift keys). This command always executes immediately.
