mathieeedoubbas.library/IEEEDPFlt ]" border=0> NAME IEEEDPFlt -- convert integer to IEEE double precision number SYNOPSIS x = IEEEDPFlt( y ); d0/d1 d0 double x; long y; FUNCTION Convert a signed 32 bit value to a double precision IEEE value and return it in d0/d1.
