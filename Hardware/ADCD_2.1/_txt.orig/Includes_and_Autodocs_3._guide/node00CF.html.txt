SYNOPSIS struct Node * AllocClickTabNode(Tag, ...) node = AllocClickTabNode(Tag, ...) struct Node * AllocClickTabNodeA(struct TagItem *) node = AllocClickTabNodeA(taglist) FUNCTION Allocates a node that can be added to the Exec linked list of labels in the clicktab. This is the only way to allocate a node for this list, you cannot allocate nodes yourself because the ClickTab class uses a private node structure. INPUTS columns - How many columns your ClickTab has. taglist - Attributes for the node, passed onto SetClickTabNodeAttrsA() . RESULT node - A node that can be added into the Exec list of labels for a ClickTab gadget. SEE ALSO FreeClickTabNode() , SetClickTabNodeAttrsA() [Back to Amiga Developer Docs ]
