icon.library/AddFreeList ]" border=0> NAME AddFreeList - add memory to a free list. SYNOPSIS status = AddFreeList(free, mem, len) D0 A0 A1 A2 BOOL AddFreeList(struct FreeList *, APTR , ULONG); FUNCTION This routine adds the specified memory to the free list. The free list will be extended (if required). If there is not enough memory to complete the call, NULL is returned. Note that AddFreeList() does NOT allocate the requested memory. It only records the memory in the free list. INPUTS free -- a pointer to a FreeList structure mem -- the base of the memory to be recorded len -- the length of the memory to be recorded RESULTS status -- TRUE if the call succeeded else FALSE; SEE ALSO exec.library/AllocEntry exec.library/FreeEntry icon.library/FreeFreeList workbench/workbench.h [Back to Amiga Developer Docs ]
