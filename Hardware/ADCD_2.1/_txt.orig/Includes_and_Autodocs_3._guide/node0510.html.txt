printer.device/PRD_QUERY ]" border=0> NAME PRD_QUERY -- query printer port/line status FUNCTION This command returns the status of the printer port's lines and registers. Since the printer port uses either the serial or parallel port for i/o, the actual status returned is either the serial or parallel port's status. IO REQUEST io_Message mn_ReplyPort set if quick I/O is not possible io_Device preset by the call to OpenDevice io_Command PRD_QUERY io_Data ptr to 2 UBYTES where result will be stored.
