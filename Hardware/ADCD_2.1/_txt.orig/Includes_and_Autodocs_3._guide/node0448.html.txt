listbrowser_gc/ListBrowserSelectAll ]" border=0> NAME ListBrowserSelectAll -- Select all nodes in a multiselect ListBrowser. SYNOPSIS VOID ListBrowserSelectAll(struct List *); ListBrowserSelectAll(list); FUNCTION Sets the LBNA_Selected bit to TRUE in all nodes in list. This is a quick way to select all nodes in a multi-select ListBrowser. Note that as with any modifications to a ListBrowser list, the list MUST be detached to the listbrowser at the time of modification. INPUTS list - a list of ListBrowser nodes, not currently attached to a ListBrowser. [Back to Amiga Developer Docs ]
