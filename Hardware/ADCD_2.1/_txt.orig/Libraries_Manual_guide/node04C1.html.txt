B / rootclass / New Methods: OM_REMMEMBER ]" border=0> Tells an object to remove a member object from its personal list. The member object should have already been added with OM_ADDMEMBER . This method uses the same custom message as OM_ADDMEMBER. Normally a dispatcher implements OM_REMMEMBER by sending the OM_REMOVE message to the opam_Object object. The return value for this method is not explicitly defined. [Back to Amiga Developer Docs ]
