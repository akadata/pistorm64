RangeRand() RangeRand() accepts a value from 1 to 65535, and returns a value within that range (a 16-bit integer). Note that this function is implemented in C. fclose() Closes a file. fgetc() Gets a character from a file. fprintf() Prints a formatted output line to a file. fputc() Puts character to file. fputs() Writes a string to file. getchar() Gets a character from stdin. printf() Puts format data to stdout. putchar() Puts character to stdout. puts() Puts a string to stdout, followed by newline. sprintf() Formats data into a string (see Exec RawDoFmt() ).
