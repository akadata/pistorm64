30 / Layers / Creating and Using New Layers ]" border=0> The functions described in this section are generally not safe to use with Intuition. To create new layers for Intuition you use Intuition window calls (see the " Intuition Windows " chapter earlier in this book). Only applications that create and mange their own View will be able to call the layer creation and updating functions discussed here.
