iffparse.library/PopChunk ]" border=0> NAME PopChunk -- Pop top context node off context stack. SYNOPSIS error = PopChunk (iff) d0 a0 LONG error; struct IFFHandle *iff; FUNCTION Pops top context chunk and frees all associated local context items. The function is normally called only for writing files and signals the end of a chunk. INPUTS iff - pointer to IFFHandle struct. RESULT error - 0 if successful or an IFFERR_#? error code if not successful. EXAMPLE NOTES BUGS SEE ALSO PushChunk() [Back to Amiga Developer Docs ]
