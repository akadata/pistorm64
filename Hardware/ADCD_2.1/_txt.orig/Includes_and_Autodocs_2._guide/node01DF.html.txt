iffparse.library/WriteChunkRecords ]" border=0> NAME WriteChunkRecords -- Write records from a buffer to the current chunk. SYNOPSIS error = WriteChunkRecords (iff, buf, recsize, numrec) d0 a0 a1 d0 d1 LONG error; struct IFFHandle *iff; UBYTE *buf; LONG recsize, numrec; FUNCTION Writes record elements from the buffer into the top chunk. This function operates much like ReadChunkBytes() . INPUTS iff - pointer to IFFHandle struct. buf - pointer to buffer area containing data. recsize - size of data records to write. numrec - number of data records to write. RESULT error - (positive) number of whole records written if successful or a (negative) IFFERR_#? error code if not successful.
