iffparse.library/CurrentChunk ]" border=0> NAME CurrentChunk -- Get context node for current chunk. SYNOPSIS top = CurrentChunk (iff) d0 a0 struct ContextNode *top; struct IFFHandle *iff; FUNCTION Returns top context node for the given IFFHandle struct. The top context node corresponds to the chunk most recently pushed on the stack, which is the chunk where the stream is currently positioned. The ContextNode structure contains information on the type of chunk currently being parsed (or written), its size and the current position within the chunk. INPUTS iff - pointer to IFFHandle struct. RESULT top - pointer to top context node or NULL if none. EXAMPLE NOTES BUGS SEE ALSO PushChunk() , PopChunk() , ParseIFF() , ParentChunk() [Back to Amiga Developer Docs ]
