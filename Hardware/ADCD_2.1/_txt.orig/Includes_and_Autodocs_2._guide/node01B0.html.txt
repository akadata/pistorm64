commodities.library/RemoveCxObj ]" border=0> NAME RemoveCxObj -- remove a commodity object from a list. (V36) SYNOPSIS RemoveCxObj(co); A0 VOID RemoveCxObj(CxObj *); FUNCTION This function remove 'co' from any list it may be a part of. Will not crash if 'co' is NULL, or if it has not been inserted in a list (and is not corrupted). It is not recommended to remove a broker from the master list. INPUTS co - the object to remove, may be NULL SEE ALSO AttachCxObj() , EnqueueCxObj() , InsertCxObj() [Back to Amiga Developer Docs ]
