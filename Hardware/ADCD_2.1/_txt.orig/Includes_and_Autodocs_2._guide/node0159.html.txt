amiga.lib/DoSuperMethodA ]" border=0> NAME DoSuperMethodA -- Perform method on object coerced to superclass. (V36) SYNOPSIS result = DoSuperMethodA( cl, obj, msg ) ULONG DoSuperMethodA( struct IClass *, Object *, Msg * ); FUNCTION Boopsi support function that invokes the supplied message on the specified object, as though it were the superclass of the specified class. INPUTS cl - pointer to boopsi class whose superclass is to receive the message obj - pointer to boopsi object msg - pointer to method-specific message to send RESULT result - class and message-specific result. NOTES While this function requires V36 or higher intuition.library, it first appeared in V37 amiga.lib. Some early example code may refer to this function as DSM().
