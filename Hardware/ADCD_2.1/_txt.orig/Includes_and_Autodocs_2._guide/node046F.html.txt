graphics.library/RemFont ]" border=0> NAME RemFont -- Remove a font from the system list. SYNOPSIS RemFont(textFont) A1 void RemFont(struct TextFont *); FUNCTION This function removes a font from the system, ensuring that access to it is restricted to those applications that currently have an active pointer to it: i.e. no new SetFont requests to this font are satisfied. INPUTS textFont - the TextFont structure to remove. RESULT BUGS SEE ALSO SetFont() AddFont() graphics/text.h [Back to Amiga Developer Docs ]
