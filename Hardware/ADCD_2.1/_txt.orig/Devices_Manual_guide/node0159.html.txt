The example code is an excellent starting point for those who want to create a custom device and add it to the Amiga's system software. The example is a complete four-unit, static-sized RAM disk that works under the old (standard) filing system, the new Fast Filing System (FFS), and has optional code to bind it to an AUTOCONFIG(TM) device.
