NAME
    NewRegion -- Get an empty region.

SYNOPSIS
    region = NewRegion()
      d0

    struct Region *NewRegion();

FUNCTION
    Create a Region structure, initialize it to empty, and return
    a pointer it.

RESULTS
    region - pointer to initialized region. If it could not allocate
            required memory region = NULL.

INPUTS
    none
