3 / All Sizes / When the Big Picture is Larger than the Display Window ]" border=0> If you design a memory picture larger than the display window, you must choose which part of it to display. Displaying a portion of a larger playfield differs in the following ways from displaying the basic playfields described up to now: * If the big picture in memory is larger than the display window, you must respecify the modulos. The modulo must be some value other than 0. * You must allocate more memory for the larger memory picture.
