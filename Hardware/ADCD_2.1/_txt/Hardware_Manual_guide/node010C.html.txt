TITLE: E / / Game Controller Interface Specification / Fire Buttons

E / / Game Controller Interface Specification / Fire Buttons ]" border=0> The fire buttons are normally opened switches routed to the 8520 adapter PRA0 as follows: PRA0 bit 7 = Fire* left controller port PRA0 bit 6 = Fire* right controller port Before reading this register, the corresponding bits of the data direction register must be cleared to define input mode: DDRA0 cleared as appropriate NOTE: ----- Do not disturb the settings of other bits in DDRA0 (Use of ROM kernel calls is recommended). Fire buttons are always valid and may be read at any time.
