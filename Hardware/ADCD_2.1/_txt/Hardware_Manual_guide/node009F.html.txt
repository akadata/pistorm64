C / New Features of the Enhanced Chip Set / New Genlock Capabilities ]" border=0> The enhanced Denise chip provides the following four new genlock features: * Chromakey * BitPlaneKey * BorderBlank * BorderNotTransparent ChromaKey allows any color register to control the video overlay. BitPlaneKey allows any bitplane to enable the video overlay. BorderBlank creates a transparent "frame" surrounding the active area. BorderNotTransparent makes an opaque "frame" surrounding the active area. [Back to Amiga Developer Docs ]
