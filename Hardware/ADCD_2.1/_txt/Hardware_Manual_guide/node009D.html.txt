C / New Features of the Enhanced Chip Set / New Mode Resolutions ]" border=0> The enhanced Denise chip provides the new SuperHires mode with up to 1280 horizontal pixels per scanline on a standard NTSC or PAL display. All of the standard display resolutions and depths of the original chip set are supported with the ECS. [Back to Amiga Developer Docs ]
