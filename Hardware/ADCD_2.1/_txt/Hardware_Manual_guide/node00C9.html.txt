4 Sprite Hardware / Summary of Sprite Registers ]" border=0> There are eight complete sets of registers used to describe the sprites. Each set consists of five registers. Only the registers for sprite 0 are described here. All of the others are the same, except for the name of the register, which includes the appropriate number.
