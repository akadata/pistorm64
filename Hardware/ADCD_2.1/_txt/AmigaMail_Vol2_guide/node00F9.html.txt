TITLE: OFNT (0x4F464E54) / ID of outline font file /

OFNT (0x4F464E54) / ID of outline font file / ]" border=0> ProVector's outline fonts are stored in an IFF FORM called OFNT. This IFF is a separate file from a DR2D. DR2D's FONS chunk refers only to fonts defined in the OFNT form. [Back to Amiga Developer Docs ]
