33 IFFParse Library / Function Reference ]" border=0> The following are brief descriptions of the IFFParse functions discussed in this chapter. IFFParse library functions are avialable in Release 2 of the Amiga OS and are backward compatible with older versions of the system. Further information about these and other IFFParse functions can be found in the 3rd edition of the Amiga ROM Kernel Reference Manual: Includes and Autodocs, also from Addison-Wesley.
