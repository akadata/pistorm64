TITLE: 27 / / The RastPort Structure / RastPort Graphics Element Pointer

27 / / The RastPort Structure / RastPort Graphics Element Pointer ]" border=0> The graphics element pointer in the RastPort structure is called GelsInfo . If you are doing graphics animation using the GELS system, this pointer must refer to a properly initialized GelsInfo structure. See the chapter on " Graphics Sprites, Bobs and Animation " for more information. [Back to Amiga Developer Docs ]
