TITLE: 28 / / Displaying the VSprites / Merging VSprite Instructions

28 / / Displaying the VSprites / Merging VSprite Instructions ]" border=0> Once DrawGList() has prepared the necessary instructions and memory areas to display the data, the VSprites are installed into the display with MrgCop() . (DrawGList() does not actually draw the VSprites, it only prepares the Copper instructions.) struct View *view; MrgCop(view); The view is a pointer to the View structure whose Copper instructions are to be merged.
