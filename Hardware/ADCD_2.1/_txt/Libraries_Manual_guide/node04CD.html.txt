TITLE: B / modelclass / Changed Methods: OM_ADDMEMBER

B / modelclass / Changed Methods: OM_ADDMEMBER ]" border=0> This method tells a model to add an object to its broadcast list. When the object disposes of itself, it will also dispose of any objects remaining on its broadcast list. The return value for this method is not explicitly defined. See the rootclass description of OM_ADDMEMBER for more information. [Back to Amiga Developer Docs ]
