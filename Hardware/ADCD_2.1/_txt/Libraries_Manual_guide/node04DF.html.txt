B Boopsi Class Reference / frameiclass ]" border=0> Class: frameiclass Superclass: imageclass Include File: imageclass.h > This is a class of framing image, which can optionally fill itself. Its purpose is to frame other display elements using an embossed or recessed rectangular frame. The frame renders itself using the appropriate DrawInfo pens (SHINEPEN, SHADOWPEN, etc.). This class is intelligent enough to bound or center its contents. New Methods: ------------ None.
