TITLE: 27 / / Data Move Operations / Setting a Whole Raster to a Color

27 / / Data Move Operations / Setting a Whole Raster to a Color ]" border=0> You can preset a whole raster to a single color by using the function SetRast() . A call to this function takes the following form: SetRast(&rastPort, pen); As always, the &rastPort is a pointer to the RastPort you wish to use. Set the pen argument to the color register you want to fill the RastPort with. [Back to Amiga Developer Docs ]
