6 / Setting Up Menus / Menu Selection Messages ]" border=0> An input event is generated every time the user activates the menu system, either by pressing the mouse menu button or its keyboard equivalent (right Amiga Alt), or entering an appropriate command key sequence. The program receives a message of type IDCMP_MENUPICK detailing which menu items or sub-items were selected. Even if the user activates the menu system without selecting a menu item or sub-item, an event is generated. Multi-Selection of Menu Items [Back to Amiga Developer Docs ]
