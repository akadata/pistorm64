TITLE: 28 / Using Virtual Sprites / Displaying the VSprites

28 / Using Virtual Sprites / Displaying the VSprites ]" border=0> The next few sections explain how to display the VSprites. The following system functions are used: DrawGList() Draws the VSprites into the current RastPort . MrgCop() Installs the VSprites into the display. LoadView() Asks the system to display the new View . WaitTOF() Synchronizes the functions with the display.
