TITLE: B / fillrectclass / Attributes: IA_APattern, IA_APatSize (IS)

B / fillrectclass / Attributes: IA_APattern, IA_APatSize (IS) ]" border=0> These attributes supply the fillrectclass object with an area fill pattern. The IA_APattern attribute points to the area fill pattern for the object. The IA_APatSize attribute is the depth of the area fill pattern. These attribute values are similar to the parameters passed to the SetAfPt() macro (defined in gfxmacros.h >) and indirectly correspond to fields in a RastPort structure. For more information on these patterns, see the section on patterns in the " Graphics Primitives " chapter of the Amiga ROM Kernel Reference Manual: Libraries. [Back to Amiga Developer Docs ]
