graphics.library/ChangeExtSpriteA ]" border=0> NAME ChangeExtSpriteA -- Change the sprite image pointer. (V39) SYNOPSIS ChangeExtSpriteA( vp, oldsprite, newsprite, tags) a0 a1 a2 a3 success=ChangeExtSpriteA(struct ViewPort *, struct ExtSprite *, struct ExtSprite *, struct TagList *); success=ChangeExtSprite(vp,old_sp,new_sp,tag,....); FUNCTION Attempt to change which sprite is displayed for a given sprite engine. INPUTS vp - pointer to ViewPort structure that this sprite is relative to, or 0 if relative only top of View oldsprite - pointer the old ExtSprite structure newsprite - pointer to the new ExtSprite structure.
