TITLE: printer.device/CMD_STOP

printer.device/CMD_STOP ]" border=0> NAME CMD_STOP -- pause current and queued I/O requests (immediate) FUNCTION CMD_STOP pauses all queued requests for the unit, and tries to pause the current I/O request. The only commands that will be subsequently allowed to be performed are immediate I/O requests, which include those to start, flush, and finish the I/O after the stop command.
