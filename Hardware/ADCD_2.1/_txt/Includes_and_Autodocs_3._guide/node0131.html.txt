disk.resource/ReadUnitID ]" border=0> NAME ReadUnitID - reread and return the type of drive (V37) SYNOPSIS idtype = ReadUnitID( unitNum ), DRResource D0 D0 A6 ULONG ReadUnitID(LONG); FUNCTION Rereads the drive id for a specific unit (for handling drives that change ID according to what sort of disk is in them.
