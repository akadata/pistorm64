TITLE: exec.library/RemDevice

exec.library/RemDevice ]" border=0> NAME RemDevice -- remove a device from the system SYNOPSIS RemDevice(device) A1 void RemDevice(struct Device *); FUNCTION This function calls the device's EXPUNGE vector, which requests that a device delete itself. The device may refuse to do this if it is busy or currently open. This is not typically called by user code. There are certain, limited circumstances where it may be appropriate to attempt to specifically flush a certain device. Example: /* Attempts to flush the named device out of memory. */ #include #include void FlushDevice(name) STRPTR name; { struct Device *result; Forbid(); if(result=(struct Device *)FindName(&SysBase->DeviceList,name)) RemDevice(result); Permit(); } INPUTS device - pointer to a device node SEE ALSO AddLibrary() [Back to Amiga Developer Docs ]
