utility.library/NamedObjectName ]" border=0> NAME NamedObjectName -- return the name of the object. (V39) SYNOPSIS name = NamedObjectName(object); D0 A0 STRPTR NamedObjectName(struct NamedObject *); FUNCTION Returns the name of the object passed in... Note that the name string is passed back as just a pointer to a read-only name. If the object goes away, so does the name. INPUTS object - the object, may be NULL in which case this function returns NULL. RESULT name - pointer to the name string, or NULL if 'object' is NULL.
