TITLE: intuition.library/OffMenu

intuition.library/OffMenu ]" border=0> NAME OffMenu -- Disable the given menu or menu item. SYNOPSIS OffMenu( Window , MenuNumber ) A0 D0 VOID OffMenu( struct Window *, UWORD ); FUNCTION This command disables a sub-item, an item, or a whole menu. This depends on the contents of the data packed into MenuNumber, which is described in the Intuition Reference Manual. INPUTS Window = pointer to the window MenuNumber = the menu piece to be disabled RESULT None BUGS SEE ALSO OnMenu() , ResetMenuStrip() [Back to Amiga Developer Docs ]
