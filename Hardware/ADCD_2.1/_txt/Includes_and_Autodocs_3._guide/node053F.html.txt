TITLE: resource.library/RL_CloseResource

resource.library/RL_CloseResource ]" border=0> NAME RL_CloseResource - close and free a resource object (V44) SYNOPSIS RL_CloseResource(resource) A0 VOID RL_CloseResource(RESOURCEFILE); FUNCTION This routine closes a resource object that was previously opened with RL_OpenResource . It frees all objects and object groups. INPUTS resource -- a pointer to a resource. A value of NULL is acceptable which means a no-op.
