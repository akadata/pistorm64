TITLE: mathffp.library/SPCeil

mathffp.library/SPCeil ]" border=0> NAME SPCeil -- Compute Ceil function of a number. SYNOPSIS x = SPCeil(y) D0 D0 float SPCeil(float y); FUNCTION Calculate the least integer greater than or equal to x and return it. This identity is true. Ceil(x) = -Floor(-x). INPUTS y - Motorola Fast Floating Point Format Number. RESULT x - Motorola Fast Floating Point Format Number. BUGS None. SEE ALSO SPFloor() [Back to Amiga Developer Docs ]
