TITLE: locale.library/CloseCatalog

locale.library/CloseCatalog ]" border=0> NAME CloseCatalog -- close a message catalog. (V38) SYNOPSIS CloseCatalog(catalog); A0 VOID CloseCatalog(struct Catalog *); FUNCTION Concludes access to a message catalog. The usage count of the catalog is decremented. When this count reaches 0, the catalog can be expunged from system memory whenever a memory panic occurs. INPUTS catalog - the message catalog to close. A NULL catalog is a valid parameter and is simply ignored. SEE ALSO OpenCatalog() , GetCatalogStr() [Back to Amiga Developer Docs ]
