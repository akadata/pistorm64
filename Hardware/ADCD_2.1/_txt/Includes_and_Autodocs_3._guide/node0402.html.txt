TITLE: intuition.library/WBenchToBack

intuition.library/WBenchToBack ]" border=0> NAME WBenchToBack -- Send the Workbench screen in back of all screens. SYNOPSIS Success = WBenchToBack() D0 BOOL WBenchToBack( VOID ); FUNCTION Causes the Workbench screen, if it's currently opened, to go behind all other screens. This does not 'move' the screen up or down, instead only affects the depth-arrangement of the screens. INPUTS None RESULT If the Workbench screen was opened, this function returns TRUE, otherwise it returns FALSE. BUGS SEE ALSO WBenchToFront() , ScreenToFront() [Back to Amiga Developer Docs ]
