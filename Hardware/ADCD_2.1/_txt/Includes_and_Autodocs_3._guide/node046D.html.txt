lowlevel.library/StartTimerInt ]" border=0> NAME StartTimerInt -- start the timer associated with the timer interrupt. (V40) SYNOPSIS StartTimerInt(intHandle, timeInterval, continuous); A1 D0 D1 VOID StartTimerInt(APTR, ULONG, BOOL); FUNCTION This routine starts a stopped timer that is assocatied with a timer interrupt created by AddTimerInt() . INPUTS intHandle - handle obtained from AddTimerInt() . timeInterval - number of micoseconds between interrupts. The maximum value allowed is 90,000. If higher values are passed there will be unexpected results. continuous - FALSE for a one shot interrupt. TRUE for multiple interrupts.
