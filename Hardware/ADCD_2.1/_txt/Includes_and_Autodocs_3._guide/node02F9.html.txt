TITLE: graphics.library/InitView

graphics.library/InitView ]" border=0> NAME InitView - Initialize View structure. SYNOPSIS InitView( view ) a1 void InitView( struct View * ); FUNCTION Initialize View structure to default values. INPUTS view - pointer to a View structure RESULT View structure set to all 0's. (1.0,1.1.1.2) Then values are put in DxOffset,DyOffset to properly position default display about .5 inches from top and left on monitor. InitView pays no attention to previous contents of view.
