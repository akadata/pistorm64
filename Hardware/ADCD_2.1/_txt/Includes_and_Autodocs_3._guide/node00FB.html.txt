console.device/CD_ASKDEFAULTKEYMAP ]" border=0> NAME CD_ASKDEFAULTKEYMAP -- get the current default keymap FUNCTION Fill the io_Data buffer with the current console device default keymap, which is used to initialize console unit keymaps when opened, and by RawKeyConvert with a null keyMap parameter. IO REQUEST io_Message mn_ReplyPort set if quick I/O is not possible io_Device preset by the call to OpenDevice io_Unit preset by the call to OpenDevice io_Command CD_ASKDEFAULTKEYMAP io_Flags IOF_QUICK if quick I/O possible, else zero io_Length sizeof(*keyMap) io_Data struct KeyMap *keyMap pointer to a structure that describes the raw keycode to byte stream conversion. RESULTS This function sets the io_Error field in the IOStdReq , and fills the structure pointed to by io_Data with the current device default key map.
