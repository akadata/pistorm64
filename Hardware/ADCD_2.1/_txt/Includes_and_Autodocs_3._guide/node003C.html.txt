TITLE: amiga.lib/FastRand

amiga.lib/FastRand ]" border=0> NAME FastRand - quickly generate a somewhat random integer SYNOPSIS number = FastRand(seed); ULONG FastRand(ULONG); FUNCTION Seed value is taken from stack, shifted left one position, exclusive-or'ed with hex value $1D872B41 and returned. INPUTS seed - a 32-bit integer RESULT number - new random seed, a 32-bit value SEE ALSO RangeRand() [Back to Amiga Developer Docs ]
