utility.library/AddNamedObject ]" border=0> NAME AddNamedObject -- add a named object to the given namespace. (V39) SYNOPSIS success = AddNamedObject(nameSpace, object); D0 A0 A1 BOOL AddNamedObject(struct NamedObject *, struct NamedObject *); FUNCTION Adds a new item to a NameSpace. If the NameSpace doesn't support duplicate names, a search for a duplicate will be made, and 0 (failure) will be returned. Otherwise, the entry will be Enqueue()ed to the NameSpace. INPUTS nameSpace - the name space to add to (NULL for root namespace) object - the object to add (If NULL, will return failure) RESULT success - whether the operation succeeded. Check this always!
