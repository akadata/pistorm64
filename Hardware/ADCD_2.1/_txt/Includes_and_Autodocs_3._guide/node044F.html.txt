TITLE: locale.library/--environment_vars--

locale.library/--environment_vars-- ]" border=0> Starting with V40, locale.library maintains a global environment variable called "Language" which contains the name of the current default language as used in the system. This is the name of the language associated with the Locale structure returned by OpenLocale(NULL).
