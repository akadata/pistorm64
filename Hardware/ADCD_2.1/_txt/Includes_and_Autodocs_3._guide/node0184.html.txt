dos.library/IsInteractive ]" border=0> NAME IsInteractive -- Discover whether a file is "interactive" SYNOPSIS status = IsInteractive( file ) D0 D1 BOOL IsInteractive(BPTR) FUNCTION The return value 'status' indicates whether the file associated with the file handle 'file' is connected to a virtual terminal.
