exec.library/ColdReboot ]" border=0> NAME ColdReboot - reboot the Amiga (V36) SYNOPSIS ColdReboot() void ColdReboot(void); FUNCTION Reboot the machine. All external memory and periperals will be RESET , and the machine will start its power up diagnostics. This function never returns. INPUT A chaotic pile of disoriented bits.
