dos.library/FreeArgs ]" border=0> NAME FreeArgs - Free allocated memory after ReadArgs() (V36) SYNOPSIS FreeArgs(rdargs) D1 void FreeArgs(struct RDArgs *) FUNCTION Frees memory allocated to return arguments in from ReadArgs() . If ReadArgs allocated the RDArgs structure it will be freed. If NULL is passed in this function does nothing. INPUTS rdargs - structure returned from ReadArgs() or NULL.
