commodities.library/CxObjType ]" border=0> NAME CxObjType -- obtain the type of a commodity object. (V36) SYNOPSIS type = CxObjType(co); D0 A0 ULONG CxObjType(CxObj *); FUNCTION This function should not really be necessary. It returns the type of a commodity object, which you should already know, since you created it in the first place. INPUTS co - the commodity object to inquire about (may be NULL) RESULTS type - the type of the commodity object, possible values are defined in . Returns CX_INVALID if 'co' is NULL.
