TITLE: layers.library/DisposeLayerInfo

layers.library/DisposeLayerInfo ]" border=0> NAME DisposeLayerInfo -- Return all memory for LayerInfo to memory pool SYNOPSIS DisposeLayerInfo( li ) a0 void DisposeLayerInfo( struct Layer_Info *); FUNCTION return LayerInfo and any other memory attached to this LayerInfo to memory allocator. Note: if you wish to delete the layers associated with this Layer_Info structure, remember to call DeleteLayer() for each of the layers before calling DisposeLayerInfo(). INPUTS li - pointer to LayerInfo structure EXAMPLE -- delete the layers associated this Layer_Info structure -- DeleteLayer(li,simple_layer); DeleteLayer(li,smart_layer); -- see documentation on DeleteLayer about deleting SuperBitMap layers -- my_super_bitmap_ptr = super_layer->SuperBitMap; DeleteLayer(li,super_layer); -- now dispose of the Layer_Info structure itself -- DisposeLayerInfo(li); BUGS SEE ALSO DeleteLayer() , graphics/layers.h [Back to Amiga Developer Docs ]
