(V40) SYNOPSIS struct Node * AllocRadioButtonNode(UWORD, Tag , ...) node = AllocRadioButtonNode(columns, Tag , ...) struct Node * AllocRadioButtonNodeA(UWORD, struct TagItem *) node = AllocRadioButtonNodeA(taglist) FUNCTION Allocates a node that can be added to the Exec linked list of labels in the radiobutton. This is the only way to allocate a node for this list, you cannot allocate nodes yourself because the RadioButton class uses a private node structure. INPUTS taglist - Attributes for the node, passed onto SetRadioButtonNodeAttrsA() . RESULT node - A node that can be added into the Exec list of labels for a RadioButton gadget.
