TITLE: diskfont.library/DisposeFontContents

diskfont.library/DisposeFontContents ]" border=0> NAME DisposeFontContents -- Free the result from NewFontContents . (V34) SYNOPSIS DisposeFontContents(fontContentsHeader) A1 VOID DisposeFontContents( struct FontContentsHeader * ); FUNCTION This function frees the array of FontContents entries returned by NewFontContents . INPUTS fontContentsHeader - a struct FontContentsHeader pointer returned by NewFontContents . EXCEPTIONS This command was first made available as of version 34. A fontContentsHeader other than one acquired by a call NewFontContents will crash. SEE ALSO NewFontContents() to get structure freed here. [Back to Amiga Developer Docs ]
