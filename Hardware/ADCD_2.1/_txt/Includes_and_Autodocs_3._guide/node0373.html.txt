TITLE: iffparse.library/PopChunk

iffparse.library/PopChunk ]" border=0> NAME PopChunk -- pop top context node off context stack. (V36) SYNOPSIS error = PopChunk(iff); D0 A0 LONG PopChunk(struct IFFHandle *); FUNCTION Pops top context chunk and frees all associated local context items. The function is normally called only for writing files and signals the end of a chunk. INPUTS iff - pointer to IFFHandle structure. RESULT error - 0 if successful or an IFFERR_#? error code if unsuccessful. SEE ALSO PushChunk() , [Back to Amiga Developer Docs ]
