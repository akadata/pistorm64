datatypes.library/RemoveDTObject ]" border=0> NAME RemoveDTObject - Remove an object from a window. (V39) SYNOPSIS position = RemoveDTObject (window, object); d0 a0 a1 LONG RemoveDTObject (struct Window *, Object *); FUNCTION Removes the object from the window's object list. This will wait until the AsyncLayout process is complete. The object will receive a DTM_REMOVEDTOBJECT message to inform the object it has been removed. INPUTS window - Pointer to the window. object - Pointer to an object as returned by NewDTObjectA() . RETURNS Returns the ordinal position of the removed object. If the object wasn't found in the appropriate list then a -1 is returned. SEE ALSO AddDTObject() , intuition.library/RemoveGList() [Back to Amiga Developer Docs ]
