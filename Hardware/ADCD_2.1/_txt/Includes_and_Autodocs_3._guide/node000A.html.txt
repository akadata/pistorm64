amigaguide.library/GetAmigaGuideAttr ]" border=0> NAME GetAmigaGuideAttr - Get an AmigaGuide attribute. (V34) SYNOPSIS retval = GetAmigaGuideAttr (tag, handle, storage); d0 d0 a0 a1 LONG GetAmigaGuideAttr (Tag, AMIGAGUIDECONTEXT , ULONG *); FUNCTION This function is used to obtain attributes from AmigaGuide. INPUTS tag - Attribute to obtain. handle - Handle to an AmigaGuide system. storage - Pointer to appropriate storage for the answer. TAGS AGA_Path (BPTR) - Pointer to the current path used by AmigaGuide. AGA_XRefList (struct List *) - Pointer to the current cross reference list.
