amiga.lib/DoSuperMethodA ]" border=0> NAME DoSuperMethodA -- Perform method on object coerced to superclass. SYNOPSIS result = DoSuperMethodA( cl, obj, msg ) ULONG DoSuperMethodA( struct IClass *, Object *, Msg ); FUNCTION Boopsi support function that invokes the supplied message on the specified object, as though it were the superclass of the specified class. INPUTS cl - pointer to boopsi class whose superclass is to receive the message obj - pointer to boopsi object msg - pointer to method-specific message to send RESULT result - class and message-specific result. NOTES This function first appears in the V37 release of amiga.lib. While it intrinsically does not require any particular release of the system software to operate, it is designed to work with the boopsi subsystem of Intuition, which was only introduced in V36.
