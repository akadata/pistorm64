TITLE: speedbar_gc/AllocSpeedButtonNodeA

speedbar_gc/AllocSpeedButtonNodeA ]" border=0> NAME AllocSpeedButtonNodeA -- Allocate a SpeedBar node. (V40) SYNOPSIS struct Node * AllocSpeedButtonNode(UWORD, Tag , ...) node = AllocSpeedButtonNode(number, Tag , ...) struct Node * AllocSpeedButtonNodeA(UWORD, struct TagItem *) node = AllocSpeedButtonNodeA(number, taglist) FUNCTION Allocates a node that can be added to the Exec linked list of labels in the speedbar. This is the only way to allocate a node for this list, you cannot allocate nodes yourself because the SpeedBar class uses a private node structure. INPUTS number - Places value in node->sbn_Node.ln_Pri, and is the numeric ID of the button within the bar. Note the current limitation, ln_Pri is a BYTE . This will be addressed, and *posibly* made obsolete and overridden by SBNA_ButtonID. taglist - Attributes for the node, passed onto SetSpeedButtonNodeAttrsA() . RESULT node - A node that can be added into the Exec list of labels for a SpeedBar gadget. SEE ALSO FreeSpeedButtonNode() , SetSpeedButtonNodeAttrsA() [Back to Amiga Developer Docs ]
