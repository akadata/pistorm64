TITLE: 15 Resources / FileSystem Resource

15 Resources / FileSystem Resource ]" border=0> The FileSystem resource returns the filesystems that are available on the Amiga. It has no functions. Opening the FileSystem resource returns a pointer to a List structure containing the current filesystems in the Amiga. Get_Filesys.c Additional programming information on the FileSystem resource can be found in the include files and the Autodocs for the FileSystem resource in the Amiga ROM Kernel Reference Manual: Includes and Autodocs and the " Expansion " chapter of the Amiga ROM Kernel Reference Manual: Libraries.
