10 / Sending Printer Commands to a Printer / Command Definitions ]" border=0> The following table describes the supported printer functions. Just Because We Have It Doesn't Mean You Do. -------------------------------------------- Not all printers support every command. Unsupported commands will either be ignored or simulated using available functions. To transmit a command to the printer device, you can either formulate a character stream containing the material shown in the "Escape Sequence" column of the table below or send an PRD_PRTCOMMAND device I/O command to the printer device with the "Name" of the function you wish to perform.
