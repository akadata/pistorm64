7 / Device Interface / Opening The Keyboard Device ]" border=0> Three primary steps are required to open the keyboard device: * Create a message port using the CreatePort() function. * Create an extended I/O request structure using the CreateExtIO() function. CreateExtIO() will initialize the I/O request with your reply port. * Open the keyboard device.
