TITLE: 11 / Device Interface / Closing The SCSI Device

11 / Device Interface / Closing The SCSI Device ]" border=0> Each OpenDevice() must eventually be matched by a call to CloseDevice() . All I/O requests must be complete before calling CloseDevice() . If any requests are still pending, abort them with AbortIO() . if (!(CheckIO(SCSIIO))) { AbortIO(SCSIIO); /* Ask device to abort any pending requests */ WaitIO(SCSIIO); /* Wait for abort, then clean up */ } CloseDevice(SCSIIO); /* Close SCSI device */ [Back to Amiga Developer Docs ]
