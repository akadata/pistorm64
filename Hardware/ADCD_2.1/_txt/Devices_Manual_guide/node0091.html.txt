4 / Reading from the Console Device / Cursor Position Report ]" border=0> If you have sent the DEVICE STATUS REPORT command sequence, the console device returns a cursor position report into your input stream. It takes the form: ; R For example, if the cursor is at column 40 and row 12, here are the ASCII values (in hex) you receive in a stream: 9B 34 30 3B 31 32 52 [Back to Amiga Developer Docs ]
