gadtools.library/FreeGadgets ]" border=0> NAME FreeGadgets -- Free a linked list of gadgets. (V36) SYNOPSIS FreeGadgets(glist) A0 VOID FreeGadgets(struct Gadget *glist); A0 FUNCTION Frees any GadTools gadgets found on the linked list of gadgets beginning with the specified one. Frees all the memory that was allocated by CreateGadgetA() . This function will return safely with no action if it receives a NULL parameter. INPUTS glist - pointer to first gadget in list to be freed. RESULT none EXAMPLE NOTES BUGS SEE ALSO CreateGadgetA() [Back to Amiga Developer Docs ]
