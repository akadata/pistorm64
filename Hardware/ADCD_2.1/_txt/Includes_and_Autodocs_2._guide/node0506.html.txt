TITLE: console.device/CMD_CLEAR

console.device/CMD_CLEAR ]" border=0> NAME CMD_CLEAR -- Clear console input buffer. FUNCTION Remove from the console input buffer any reports waiting to satisfy read requests. IO REQUEST INPUT io_Message mn_ReplyPort set if quick I/O is not possible io_Device preset by the call to OpenDevice io_Unit preset by the call to OpenDevice io_Command CMD_CLEAR io_Flags IOB_QUICK set if quick I/O is possible, else 0 SEE ALSO exec/io.h , devices/console.h [Back to Amiga Developer Docs ]
