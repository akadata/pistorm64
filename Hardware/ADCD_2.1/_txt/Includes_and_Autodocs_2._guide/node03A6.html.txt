layers.library/UnlockLayer ]" border=0> NAME UnlockLayer -- Unlock layer and allow graphics routines to use it. SYNOPSIS UnlockLayer( l ) a0 void UnlockLayer( struct Layer *); FUNCTION When finished changing the ClipRects or whatever you were doing with this layer you must call UnlockLayer() to allow other tasks to proceed with graphic output to the layer.
