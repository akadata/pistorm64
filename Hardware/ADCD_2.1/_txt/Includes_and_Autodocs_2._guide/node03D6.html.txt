(V36) SYNOPSIS tag = FindTagItem( tagVal, tagList) D0 D0 A0 struct TagItem *FindTagItem( Tag tagVal, struct TagItem *tagList ); FUNCTION Scans a TagItem "List", which is in fact a chain of arrays of TagItem structures as defined in utility/tagitem.h . Returns a pointer to the FIRST item with ti_Tag matching the 'TagVal' parameter. INPUTS tagVal - Tag value to search for. tagList - beginning of TagItem list to scan. RESULT Returns a pointer to the item with ti_Tag matching 'TagVal'. Returns NULL if there is no match or if TagList is NULL.
