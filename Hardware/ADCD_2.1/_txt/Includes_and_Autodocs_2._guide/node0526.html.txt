TITLE: trackdisk.device/TD_EJECT

trackdisk.device/TD_EJECT ]" border=0> NAME TD_EJECT -- eject the disk in the drive, if possible. FUNCTION This command causes the drive to attempt to eject the disk in it, if any. Note that the current trackdisk.device does not implement this command, but it might in the future, and other trackdisk-compatible drivers may implement this command.
