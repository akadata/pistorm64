TITLE: amiga.lib/ArgArrayDone

amiga.lib/ArgArrayDone ]" border=0> NAME ArgArrayDone -- release the memory allocated by a previous call to ArgArrayInit() . (V36) SYNOPSIS ArgArrayDone(); VOID ArgArrayDone(VOID); FUNCTION This function frees memory and does cleanup required after a call to ArgArrayInit() . Don't call this until you are done using the ToolTypes argument strings. SEE ALSO ArgArrayInit() [Back to Amiga Developer Docs ]
