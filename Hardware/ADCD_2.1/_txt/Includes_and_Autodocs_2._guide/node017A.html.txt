TITLE: debug.lib/KPutStr

debug.lib/KPutStr ]" border=0> NAME KPutStr - put a string to the console (defaults to the serial port at 9600 baud) SYNOPSIS KPutStr(string) A0 FUNCTION put a null terminated string to the console. This function will not return until the string has been completely transmitted. INPUTS KPutStr is the assembly interface, a string pointer must be in A0. _KPutStr and _kputs are the C interfaces, the string pointer must be on the stack. [Back to Amiga Developer Docs ]
