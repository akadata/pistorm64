BOOL LayoutMenus(struct Menu *, APTR, Tag, ...); FUNCTION Lays out all the menus, menu items and sub-items in the supplied menu according to the supplied visual information and tag parameters. This routine attempts to columnize and/or shift the MenuItems in the event that a menu would be too tall or too wide. INPUTS menu - Pointer to menu obtained from CreateMenusA() . vi - Pointer returned by GetVisualInfoA() . taglist - Pointer to a TagItem list. TAGS GTMN_TextAttr (struct TextAttr *) - Text Attribute to use for menu-items and sub-items. If not supplied, the screen's font will be used. This font must be openable via OpenFont() when this function is called. RESULT success - TRUE if successful, false otherwise (signifies that the TextAttr wasn't openable). EXAMPLE NOTES When using this function, there is no need to also call LayoutMenuItemsA() . BUGS If a menu ends up being wider than the whole screen, it will run off the right-hand side.
