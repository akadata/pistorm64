TITLE: graphics.library/SetBPen

graphics.library/SetBPen ]" border=0> NAME SetBPen -- Set secondary pen for a RastPort SYNOPSIS SetBPen( rp, pen ) a1 d0 void SetBPen( struct RastPort *, UBYTE ); FUNCTION Set the secondary drawing pen for lines, fills, and text. INPUTS rp - pointer to RastPort structure. pen - (0-255) RESULT Changes the minterms in the RastPort to reflect new secondary pen. Sets line drawer to restart pattern.
