(V36) SYNOPSIS GT_EndRefresh(win, complete) A0 D0 VOID GT_EndRefresh(struct Window *, BOOL complete); FUNCTION Invokes the intuition.library/EndRefresh() function in a manner friendly to the Gadget Toolkit. This function call permits GadTools gadgets to refresh themselves at the correct time. Call this function to EndRefresh() when you have used GT_BeginRefresh() . INPUTS win - Pointer to Window structure for which a IDCMP_REFRESHWINDOW IDCMP event was received. complete - TRUE when done with refreshing. RESULT None.
