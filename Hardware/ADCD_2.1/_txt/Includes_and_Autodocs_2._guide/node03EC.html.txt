gadtools.library/FreeMenus ]" border=0> NAME FreeMenus -- Frees memory allocated by CreateMenusA() . (V36) SYNOPSIS FreeMenus(menu) A0 VOID FreeMenus(struct Menu *); FUNCTION Frees the menus allocated by CreateMenusA() . It is safe to call this function with a NULL parameter. INPUTS menu - Pointer to menu structure (or first MenuItem) obtained from CreateMenusA() . RESULT None. EXAMPLE NOTES BUGS SEE ALSO CreateMenusA() [Back to Amiga Developer Docs ]
