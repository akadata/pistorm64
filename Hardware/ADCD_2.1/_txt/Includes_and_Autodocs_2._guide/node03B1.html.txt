expansion.library/AllocConfigDev ]" border=0> NAME AllocConfigDev - allocate a ConfigDev structure SYNOPSIS configDev = AllocConfigDev() D0 FUNCTION This routine returns the address of a ConfigDev structure. It is provided so new fields can be added to the structure without breaking old, existing code. The structure is cleared when it is returned to the user. INPUTS RESULTS configDev - either a valid ConfigDev structure or NULL. EXCEPTIONS SEE ALSO FreeConfigDev BUGS [Back to Amiga Developer Docs ]
