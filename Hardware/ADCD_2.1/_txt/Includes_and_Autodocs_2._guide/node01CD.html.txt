TITLE: iffparse.library/LocalItemData

iffparse.library/LocalItemData ]" border=0> NAME LocalItemData -- Get pointer to user data for local context item. SYNOPSIS data = LocalItemData (lci) d0 a0 UBYTE *data; struct LocalContextItem *lci; FUNCTION Returns pointer to the user data associated with the given local context item. The size of the data area depends on the "usize" argument used when allocating this item. If the pointer to the item given (lci) is NULL, the function also returns NULL. INPUTS lci - pointer to local context item or NULL. RESULT data - pointer to user data area or NULL if lci is NULL. EXAMPLE NOTES BUGS Currently, there is no way to determine the size of the user data area; you have to 'know'. SEE ALSO AllocLocalItem() , FreeLocalItem() [Back to Amiga Developer Docs ]
