TITLE: graphics.library/AskFont

graphics.library/AskFont ]" border=0> NAME AskFont -- get the text attributes of the current font SYNOPSIS AskFont(rp, textAttr) A1 A0 void AskFont(struct RastPort *, struct TextAttr *); FUNCTION This function fills the text attributes structure with the attributes of the current font in the RastPort . INPUTS rp - the RastPort from which the text attributes are extracted textAttr - the TextAttr structure to be filled. Note that there is no support for a TTextAttr . RESULT The textAttr structure is filled with the RastPort's text attributes. BUGS SEE ALSO graphics/text.h [Back to Amiga Developer Docs ]
