TITLE: narrator.device/AbortIO

narrator.device/AbortIO ]" border=0> NAME AbortIO - Abort an IO request SYNOPSIS AbortIO(IORequest) A1 FUNCTION Exec library call to abort a specified READ or WRITE request. The IORequest may be in the queue or currently active. If currently active, the request is immediately stopped and then removed. INPUTS Pointer to the IORequest block to be aborted. RESULTS io_Error field in the IORequest block set to #IOERR_ABORTED. SEE ALSO [Back to Amiga Developer Docs ]
