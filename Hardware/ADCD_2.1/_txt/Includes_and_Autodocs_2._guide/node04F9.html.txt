TITLE: timer.device/CmpTime

timer.device/CmpTime ]" border=0> NAME CmpTime -- Compare two timeval structures. SYNOPSIS result = CmpTime( Dest, Source ) D0 A0 A1 LONG CmpTime( struct timeval *, struct timeval *); FUNCTION This routine compares timeval structures A0 and A1 will be left unchanged INPUTS Dest, Source -- pointers to timeval structures. RESULTS result will be 0 if Dest has same time as source -1 if Dest has more time than source +1 if Dest has less time than source NOTES This function may be called from interrupts. SEE ALSO timer.device/AddTime() , timer.device/SubTime() BUGS Older version of this document had the sense of the return codes wrong; the code hasn't changed but the document has.
