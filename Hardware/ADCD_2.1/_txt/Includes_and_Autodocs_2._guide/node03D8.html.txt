utility.library/GetTagData ]" border=0> NAME GetTagData -- Obtain data corresponding to Tag. (V36) SYNOPSIS value = GetTagData(tagVal, default, tagList) D0 D0 D1 A0 ULONG GetTagData(Tag TagVal, ULONG Default, struct TagItem *TagList) FUNCTION Searches a TagItem list for a matching Tag value, and returns the corresponding ti_Data value for the TagItem found. If none found, will return the value passed it as 'default'. INPUTS tagVal - Tag value to search for. default - value to be returned if tagVal is not found. tagList - the TagItem list to search. RESULT value - The ti_Data value for first matching TagItem , or 'default' if a ti_Tag matching 'Tag' is not found.
