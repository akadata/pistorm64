TITLE: dos.library/MakeDosEntry

dos.library/MakeDosEntry ]" border=0> NAME MakeDosEntry -- Creates a DosList structure (V36) SYNOPSIS newdlist = MakeDosEntry(name, type) D0 D1 D2 struct DosList *MakeDosEntry(STRPTR, LONG) FUNCTION Create a DosList structure, including allocating a name and correctly null-terminating the BSTR. It also sets the dol_Type field, and sets all other fields to 0. This routine should be eliminated and replaced by a value passed to AllocDosObject()! INPUTS name - name for the device/volume/assign node. type - type of node. RESULT newdlist - The new device entry or NULL.
