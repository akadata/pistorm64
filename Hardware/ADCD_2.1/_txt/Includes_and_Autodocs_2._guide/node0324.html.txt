exec.library/AddDevice ]" border=0> NAME AddDevice -- add a device to the system SYNOPSIS AddDevice(device) A1 void AddDevice(struct Device *); FUNCTION This function adds a new device to the system device list, making it available to other programs. The device must be ready to be opened at this time.
