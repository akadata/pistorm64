TITLE: intuition.library/RemoveClass

intuition.library/RemoveClass ]" border=0> NAME RemoveClass -- Make a public boopsi class unavailable. (V36) SYNOPSIS RemoveClass( classPtr ) A0 VOID RemoveClass( struct IClass * ); FUNCTION Makes a public class unavailable for public consumption. It's OK to call this function for a class which is not yet in the internal public class list, or has been already removed. INPUTS ClassPtr = pointer to *public* class created by MakeClass() , may be NULL. RESULT None. NOTES BUGS SEE ALSO MakeClass() , FreeClass() , AddClass() Document "Basic Object-Oriented Programming System for Intuition" and the "boopsi Class Reference" document.
