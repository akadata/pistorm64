To implement mutual exclusion, gadgets must be hit-select (not
GACT_TOGGLESELECT) boolean gadgets, with the GACT_IMMEDIATE activation
type (never GACT_RELVERIFY).  All state changes must be executed upon
receiving the IDCMP_GADGETDOWN message for the gadgets.  Failure to do
this could introduce subtle out-of-phase imagery problems.
