In the Intuition system, the mouse is the normal method of making
selections and the keyboard is used for entering character data.  This
section describes how users employ the mouse to interact with the system
and how to arrange for a program to use the mouse.  It also describes the
use of the keyboard, both as a character input device and as an alternate
method of controlling the mouse pointer.
