NAME
	IEEESPAsin -- compute the arcsine of a number

   SYNOPSIS
	  x   = IEEESPAsin(  y  );
	d0	           d0

	float	x,y;

   FUNCTION
	Compute the arc sine of y in IEEE single precision

   INPUTS
	y - IEEE single precision floating point value

   RESULT
	x - IEEE single precision floating point value
