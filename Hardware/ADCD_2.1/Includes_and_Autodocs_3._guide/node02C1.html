NAME 
       ClearRegion -- Remove all rectangles from region.

   SYNOPSIS 
       ClearRegion(region)
                     a0

	viod ClearRegion( struct Region * );

   FUNCTION 
       Clip away all rectangles in the region leaving nothing.

   INPUTS 
       region - pointer to Region structure
