NAME
	GetDTString - Obtain a DataTypes string.                (V39)

    SYNOPSIS
	str = GetDTString (id);
	d0		   d0

	STRPTR GetDTString (ULONG id);

    FUNCTION
	This function is used to obtain a pointer to a localized
	DataTypes string.

    INPUTS
	id - ID of the string to obtain.

    RETURNS
	A pointer to a NULL terminated string.
