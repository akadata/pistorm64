NAME
	ScreenToFront -- Make the specified screen the frontmost.

    SYNOPSIS
	ScreenToFront( Screen )
		       A0

	VOID ScreenToFront( struct Screen * );

    FUNCTION
	Brings the specified Screen to the front of the display.

    INPUTS
	Screen = a pointer to a Screen structure

    RESULT
	None

    BUGS

    SEE ALSO
