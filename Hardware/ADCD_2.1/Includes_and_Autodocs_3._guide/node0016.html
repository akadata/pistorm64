NAME
        UnlockAmigaGuideBase - Unlock an AmigaGuide client.     (V34)

    SYNOPSIS
        UnlockAmigaGuideBase (key);
                              d0

        VOID UnlockAmigaGuideBase (LONG);

    FUNCTION
        This function is used to release a lock obtained with
        LockAmigaGuideBase().

    INPUTS
        key - Value returned by LockAmigaGuideBase().
