NAME
	IEEESPLog10 -- compute logarithm base 10 of a number

   SYNOPSIS
	  x   = IEEESPLog10(  y  );
	d0	            d0

	float	x,y;

   FUNCTION
	Compute the logarithm base 10 of y in IEEE single precision

   INPUTS
	y - IEEE single precision floating point value

   RESULT
	x - IEEE single precision floating point value
