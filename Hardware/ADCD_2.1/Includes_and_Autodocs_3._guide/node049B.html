NAME
	IEEEDPTieee -- convert IEEE double to IEEE single

   SYNOPSIS
	  x   = IEEEDPTieee(  y  );
	 d0	            d0/d1

	double	y;
	float   x;

   FUNCTION
	Convert IEEE double precision number to IEEE single precision.

   INPUTS
	y - IEEE double precision floating point value

   RESULT
	x - IEEE single precision floating point value
